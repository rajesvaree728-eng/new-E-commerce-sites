This E‑Commerce website is a frontend project created to practice and showcase modern web development skills. It includes multiple categories like Men, Women, Footwear, Accessories, Ethnic Wear, and Trending Fashion.

All products are loaded dynamically using a custom products.json API. This helps demonstrate real‑world concepts like data fetching, filtering, dynamic rendering, and category‑based product pages.

The goal of this project is to build a clean, responsive, and functional shopping interface that looks professional and job‑ready.

Tecdhnologies Used

  • HTML5 - page structure and layout.
  • CSS3 - Styling, grid layout, responsive design
  • JavaScript - Fetch API, dynamic product rendering
  • JSON - Custom product data API
  • Live Server - Local development environment

This project is part of my learning journey as a frontend developer, and I will continue improving it by adding features like search, filters, cart system, and checkout page.
