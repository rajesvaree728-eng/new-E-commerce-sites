Welcome to our Help section. If you are facing any issues while browsing or using this website, you can find quick solutions below.

How to Browse Products

  • Go to the category you want (Men, Women, Footwear, Accessories, Ethnic, Trending).
  • Products will load automatically from our JSON API.
  • Click on any product to view details (if detail page is added).

Images Not Loading?

Solution: Always open the website using Live Server.

Opening directly in Chrome (file:///C:/...) will block JSON and images.

Facing Slow Loading?

  • Check your internet connection.
  • Make sure images are inside the correct folder.
  • Ensure file names match exactly with JSON.

Common Errors & Fixes

  • CORS Error: Use Live Server.
  • 404 Image Not Found: Check image name and folder.
  • JSON Not Loading: Check file path and spelling.
